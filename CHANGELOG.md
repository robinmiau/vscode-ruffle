# Change Log

All notable changes to the "Ruffle" extension will be documented in this file.

## [0.0.4](https://github.com/robinmiau/vscode-ruffle/compare/v0.0.1...v0.0.4) (2025-10-04)

### Changed
- Updated `@ruffle-rs/ruffle` to `0.2.0-nightly.2025.10.1`
- Updated development dependencies including TypeScript ESLint, webpack, and Node.js types

### Fixed
- Pinned `@types/vscode` to version 1.103.0 to prevent compatibility issues

## 0.0.1 (2025-08-24)

### Added
- Initial release of the Ruffle extension for Visual Studio Code
- Support for previewing SWF files using Ruffle Flash emulator
- Automatic dependency updates via Dependabot
- Monthly automated dependency update schedule
